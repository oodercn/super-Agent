================================================================================
                    Ooder Nexus 2.0 - OpenWrt 预览版
================================================================================

版本信息
--------
版本号: 2.0-OpenWrt-Preview
发布日期: 2026-02-11
状态: OpenWrt 预览版 (OpenWrt Preview Release)

重要说明
--------
1. 此版本为 OpenWrt 预览版，专注于 OpenWrt 路由器设备管理功能
2. 预览版仅供测试和评估使用，可能包含未完善的功能和已知问题
3. 不建议在生产环境中使用预览版
4. 此版本主要针对 OpenWrt 路由器的连接、配置和管理功能

系统要求
--------
- Windows 10/11 64位操作系统
- Java 8 或更高版本 (JDK/JRE)
- 至少 2GB 可用内存
- 至少 100MB 磁盘空间
- OpenWrt 路由器设备（用于真实设备模式测试）

OpenWrt 功能特性
----------------
- OpenWrt 路由器自动发现与连接
- 路由器状态监控与实时数据显示
- 网络接口配置管理
- DHCP 服务器配置
- 防火墙规则管理
- 系统日志查看
- 远程命令执行
- 文件传输与管理

安装步骤
--------
1. 确保已安装Java 8或更高版本
   在命令行运行: java -version

2. 解压安装包到任意目录
   建议路径: C:\Program Files\OoderNexus\

3. 运行 start.bat 启动服务
   双击 start.bat 或在命令行中运行

4. 访问控制台
   打开浏览器访问: http://localhost:8081/console/index.html

5. 添加 OpenWrt 设备
   在控制台中进入"OpenWrt管理"页面，添加您的路由器设备

配置说明
--------
- 默认端口: 8081
- Mock模式: 已关闭 (使用真实设备模式)
- 配置文件: application.yml (在jar包内)
- OpenWrt 默认连接端口: 22 (SSH)

注意事项
--------
1. 首次启动可能需要一些时间初始化
2. 确保端口8081未被其他程序占用
3. 防火墙可能需要允许Java程序访问网络
4. 连接 OpenWrt 设备需要正确的IP地址、用户名和密码
5. 确保 OpenWrt 设备已开启 SSH 访问权限
6. 预览版可能会有功能变更，请关注官方更新

已知限制
--------
- 预览版仅支持部分 OpenWrt 版本测试
- 某些高级功能可能需要特定固件支持
- 多设备并发管理功能仍在优化中

技术支持
