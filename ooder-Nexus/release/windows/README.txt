ooderNexus 2.0-openwrt-preview
Windows 版本

============================================
系统要求
============================================
- Windows 7/8/10/11
- Java 8 或更高版本
- 内存: 至少 2GB RAM
- 磁盘空间: 至少 100MB 可用空间

============================================
快速开始
============================================
1. 确保已安装 Java 8 或更高版本
   在命令提示符中运行: java -version

2. 双击运行 start.bat 启动服务

3. 打开浏览器访问: http://localhost:8081/console/index.html
