1. 角色概述

小局域网管理员是 MCP Agent 的中级使用角色，主要面向小型办公网络或企业分支机构，提供更全面的网络管理功能。

3. 页面功能说明

3.1 仪表盘 (dashboard.html)

  • 功能：展示系统整体状态和关键指标
  • 主要信息：系统运行状态、设备在线数量、安全状态概览
  • 操作：快速访问各功能模块

3.2 家庭功能相关页面

  • 家庭仪表盘 (home/home-dashboard.html)
  • 设备列表 (home/device-list.html)
  • 设备控制 (home/device-control.html)
  • 安全状态 (home/security-status.html)

3.3 局域网仪表盘 (lan/lan-dashboard.html)

  • 功能：专用于局域网环境的综合监控面板
  • 主要信息：网络拓扑、设备分布、流量统计
  • 操作：查看详细网络信息、快速故障定位

3.4 网络设备列表 (lan/network-devices.html)

  • 功能：管理局域网中的所有网络设备
  • 主要信息：设备名称、类型、IP地址、MAC地址、状态、厂商信息
  • 操作：添加设备、删除设备、查看设备详情、修改设备设置

3.5 设备详情 (lan/device-details.html)

  • 功能：查看网络设备的详细信息
  • 主要信息：设备硬件信息、网络配置、接口状态、性能指标
  • 操作：修改设备配置、诊断设备问题、更新设备固件

3.6 网络设置 (lan/network-settings.html)

  • 功能：配置局域网的基本网络参数
  • 主要信息：IP地址范围、子网掩码、网关、DNS服务器
  • 操作：修改网络参数、应用配置变更

3.7 IP管理 (lan/ip-management.html)

  • 功能：管理局域网内的IP地址分配
  • 主要信息：IP地址池、已分配IP、IP地址状态
  • 操作：分配静态IP、释放IP地址、设置IP保留

3.8 网络状态 (lan/network-status.html)

  • 功能：监控局域网的整体网络状态
  • 主要信息：连接状态、丢包率、延迟、带宽使用率
  • 操作：查看详细网络指标、诊断网络问题

3.9 带宽监控 (lan/bandwidth-monitor.html)

  • 功能：监控局域网的带宽使用情况
  • 主要信息：实时带宽使用、历史流量统计、流量峰值
  • 操作：设置带宽限制、查看流量趋势分析

4. 操作指南

4.1 网络设备管理操作

  1. 查看网络设备列表：点击「小局域网功能」→「网络设备」→「设备列表」
  2. 查看设备详情：在设备列表中选择设备，点击「详情」按钮
  3. 修改设备配置：在设备详情页面，点击「配置」选项卡进行修改

4.2 网络配置操作

  1. 修改网络设置：点击「小局域网功能」→「网络配置」→「网络设置」
  2. 管理IP地址：点击「小局域网功能」→「网络配置」→「IP管理」
  3. 应用配置变更：修改配置后点击「应用」按钮，等待配置生效

4.3 网络监控操作

  1. 查看网络状态：点击「小局域网功能」→「网络监控」→「网络状态」
  2. 监控带宽使用：点击「小局域网功能」→「网络监控」→「带宽监控」
  3. 分析流量数据：在带宽监控页面查看历史流量图表，分析流量趋势

6. 常见问题

6.1 网络设备无法发现

原因：设备未开启SNMP、网络连接问题或权限不足

解决：确保设备已开启SNMP，检查网络连接，确认设备访问权限

6.2 IP地址冲突

原因：多个设备使用相同IP地址或IP地址分配策略问题

解决：使用IP管理功能检测冲突，重新分配IP地址，优化DHCP设置

6.3 网络带宽异常

原因：存在流量异常的设备或网络攻击

解决：使用带宽监控功能定位异常设备，检查网络安全状态
