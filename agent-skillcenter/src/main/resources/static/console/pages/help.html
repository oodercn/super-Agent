    用户指南

    平台介绍

    SkillCenter 是一个技能管理和执行平台，支持技能的注册、执行、发布和管理。

    核心功能

    • 技能管理：注册、编辑、删除技能
    • 技能执行：执行技能并查看结果
    • 市场管理：发布和下载技能
    • 存储管理：备份和恢复数据
    • 系统管理：监控和控制系统

    使用流程

    1. 登录系统
    2. 注册新技能或从市场下载技能
    3. 配置技能参数
    4. 执行技能并查看结果
    5. 管理技能和查看执行历史

    常见问题

    技能管理

    如何注册新技能？

    在技能管理页面，点击"注册技能"按钮，填写技能信息并提交。

    如何发布技能到市场？

    在技能管理页面，选择要发布的技能，点击"发布到市场"按钮。

    技能执行

    如何执行技能？

    在执行管理页面，选择要执行的技能，填写执行参数并点击"执行"按钮。

    如何查看执行历史？

    在执行管理页面，切换到"执行历史"标签页查看执行记录。
