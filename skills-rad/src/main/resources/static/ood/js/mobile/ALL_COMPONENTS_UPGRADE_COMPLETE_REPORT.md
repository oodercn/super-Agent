## 升级策略总结

### 继承体系分类
1. **表单控件**: `["ood.UI", "ood.absValue"]`
   - Switch, Button, Input 等输入控件
   - 支持值管理、表单字段、验证

2. **数据集合**: `["ood.UI", "ood.absList", "ood.absValue"]`
   - List, Picker 等列表选择组件
   - 支持列表项管理 + 值管理

3. **导航组件**: 根据复杂度选择
   - TabBar: `["ood.UI", "ood.absList", "ood.absValue"]` (复杂导航)
   - Drawer: `["ood.UI", "ood.absList"]` (列表导航)
   - NavBar: `["ood.UI"]` (简单导航)

4. **显示/容器组件**: `["ood.UI"]`
   - Avatar, Badge, Card, Modal, Toast, Layout, Panel, Grid 等
   - 主要用于展示和容器功能

5. **选择组件**: `["ood.UI", "ood.absList"]`
   - ActionSheet 等提供选择功能的组件
