# OODER开源企业SuperAgent：企业级AI能力协作框架

## 项目概述

SuperAgent是一个企业级AI能力协作框架，采用分层架构设计，支持将各类设备（包括边缘设备）作为SKILL部署环境，实现了灵活的agent协作和skill管理机制。系统核心层明确包含五个中心：Auth认证中心、DataServer数据中心、Skill能力管理中心、LLM调度中心和Skillflow调度中心，各中心职责清晰，松耦合设计便于扩展和维护。

### 2. 核心中心说明

#### 2.1 Auth认证中心
- **功能**：负责系统的认证和授权管理
- **职责**：用户认证、设备认证、权限管理、安全访问控制

#### 2.2 DataServer数据中心
- **功能**：管理系统的所有数据
- **职责**：数据存储、查询、备份、恢复、数据一致性保障

#### 2.3 Skill能力管理中心
- **功能**：管理所有skill的生命周期
- **职责**：skill注册、分类、发现、状态监控、1:n Capability关系定义

#### 2.4 LLM调度中心
- **功能**：负责大语言模型的调度和管理
- **职责**：模型选择、请求分发、响应处理、多模型集成

#### 2.5 Skillflow调度中心
- **功能**：管理服务流的定义和执行
- **职责**：服务流解析、步骤调度、状态管理、异常处理、结果汇总

## 主要特点

1. **模块化设计**：各层职责清晰，便于扩展和维护
2. **松耦合架构**：组件间通过接口通信，降低依赖关系
3. **灵活的Skill管理**：支持SKILL的1:n Capability关系，将各类设备作为SKILL部署环境
4. **强大的Agent协作**：MCP/Route/End Agent三级架构，支持mesh组网
5. **完善的安全机制**：基于认证中心的安全管理，支持双向认证和加密通信
6. **可扩展的LLM集成**：通过LLM调度中心支持多种大语言模型的集成
7. **灵活的部署方式**：支持单节点和分布式部署
