## 四、术语定义整合

### 4.1 Agent层术语整合

#### MCP Agent

| 属性 | 值 |
|------|-----|
| 英文 | MCP Agent / Master Control Program Agent |
| 中文 | 主控智能体 |
| 缩写 | MCP |
| 定义（v0.6.x） | 服务流入口，管理和协调其他Agent |
| 定义（v0.7.0） | Agent层级结构的顶层节点，负责资源管理、任务调度、安全认证和全局协调 |
| 变更 | 扩展定义，明确职责范围 |

#### Route Agent

| 属性 | 值 |
|------|-----|
| 英文 | Route Agent |
| 中文 | 路由智能体 |
| 缩写 | RA |
| 定义（v0.6.x） | 负责桥接和转发，连接MCP Agent和End Agent |
| 定义（v0.7.0） | Agent层级结构的中间层节点，负责消息路由、场景组管理、技能协调 |
| 新增 | PRIMARY/BACKUP角色定义 |
| 变更 | 扩展定义，新增故障切换支持 |

#### End Agent

| 属性 | 值 |
|------|-----|
| 英文 | End Agent |
| 中文 | 终端智能体 |
| 缩写 | EA |
| 定义（v0.6.x） | 独立功能模块，执行具体任务 |
| 定义（v0.7.0） | Agent层级结构的终端节点，直接与用户交互，调用技能提供的能力 |
| 变更 | 扩展定义，明确用户交互职责 |

### 4.2 技能层术语整合

#### Skill

| 属性 | 值 |
|------|-----|
| 英文 | Skill |
| 中文 | 技能 |
| 缩写 | - |
| 定义（v0.6.x） | 物理程序，具有唯一标识，可被管理和控制，提供特定功能 |
| 定义（v0.7.0） | 场景的具体实现，包含程序代码、配置、资源等，是可独立部署和运行的功能单元 |
| 变更 | 扩展定义，明确与场景的关系 |

#### SkillPackage（新增）

| 属性 | 值 |
|------|-----|
| 英文 | Skill Package |
| 中文 | 技能包 |
| 缩写 | SP |
| 定义 | 技能的分发单元，包含技能运行所需的所有文件，可打包为ZIP或TAR.GZ格式 |
| 版本 | v0.7.0新增 |

#### SkillSource（新增）

| 属性 | 值 |
|------|-----|
| 英文 | Skill Source |
| 中文 | 技能源 |
| 缩写 | - |
| 定义 | 技能的来源地址，支持多种协议，是技能发现和下载的入口 |
| 版本 | v0.7.0新增 |

#### SkillCenter（新增）

| 属性 | 值 |
|------|-----|
| 英文 | Skill Center |
| 中文 | 技能中心 |
| 缩写 | SC |
| 定义 | 中心化的技能管理和分发服务，提供技能注册、发现、版本管理、远程部署等功能 |
| 版本 | v0.7.0新增 |

### 4.3 场景层术语整合

#### Scene

| 属性 | 值 |
|------|-----|
| 英文 | Scene |
| 中文 | 场景 |
| 缩写 | - |
| 定义（v0.6.x） | 基于特定业务需求的Agent协作模式定义 |
| 定义（v0.7.0） | 能力需求的抽象定义，描述"做什么"，是技能的契约和接口 |
| 新增 | 主场景/协作场景分类 |
| 变更 | 扩展定义，明确与技能的关系 |

#### SceneGroup（新增，原Group）

| 属性 | 值 |
|------|-----|
| 英文 | Scene Group |
| 中文 | 场景组 |
| 缩写 | SG |
| 定义（v0.6.x） | Group：为实现特定Scene而形成的Agent集合 |
| 定义（v0.7.0） | SceneGroup：共享KEY和VFS资源的Agent集合，实现场景的协作和故障切换 |
