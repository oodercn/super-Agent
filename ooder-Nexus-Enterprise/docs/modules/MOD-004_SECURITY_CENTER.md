## 1. 模块概述

### 1.1 场景定义

安全中心模块提供密钥管理和安全通信能力：

| 场景 | 描述 | 用户角色 |
|------|------|---------|
| **密钥管理** | 生成和管理密钥对 | 管理员 |
| **数据加密** | 加密敏感数据 | 系统 |
| **数字签名** | 数据完整性验证 | 系统 |
| **Token 管理** | 生成和验证 Token | 系统 |
| **场景组安全** | 多 Agent 安全通信 | 开发者 |
